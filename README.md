## Installation

Using [npm](https://www.npmjs.com):

```
npm install eslint-config-saya
```


## Documentation

Extend your [ESLint configuration](http://eslint.org/docs/user-guide/configuring.html):

```js
{
  extends: ['saya']
}
```

Available configurations:
- `saya`
- `saya/babel`
- `saya/flow`
